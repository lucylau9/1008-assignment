Accessibility

I have made this website with:

  • A simple colour scheme with high contrast between text and backgrounds
  • Text alternatives for images
  • Simple animations that are not too overwhelming
  • Text large enough to read on mobile screens

Unfortunately, this website is not fully accessible. The main reason is the game demo. Since it relies on looking at images, using a text alternative will not provide the player with the full experience.
